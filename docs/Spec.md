# 功能規格 (Spec.md)
* **核心功能**：實現雙層選單切換與多媒體播放控制。
* **防呆機制**：
    * **異常處理**：若第三方插件 (ZFBrowser) 缺失，系統需具備 Null Check 避免崩潰。
    * **邊界條件**：ESC 鍵觸發選單時，必須同步切換 `Time.timeScale` 以防背景邏輯在暫停時出錯。
### 2.1 番茄鐘 (Pomodoro)
* **規格需求**：提供 25 分鐘專注與 5 分鐘休息切換。
* **防呆機制**：當選單關閉且 `Time.timeScale` 恢復時，計時器須精準繼續，不可跳秒。

### 2.2 待辦清單 (TodoList)
* **規格需求**：支援輸入文字新增任務，並具備勾選刪除功能。
* **數據持久化**：預計透過 PlayerPrefs 儲存當前清單狀態。

### 2.3 系統時鐘 (Clock)
* **規格需求**：同步顯示系統當前時間（時:分:秒），每秒更新一次。

### 2.4 互動日曆 (Calendar)
* **規格需求**：顯示當前月份與日期，並醒目標示今日日期。
